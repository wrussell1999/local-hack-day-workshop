Get Setup with the [Discord API here](discord.md)

Get Setup with the [Reddit API here](reddit.md)

## Development

```bash
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

## Run

```bash
(.venv) $ python -m discord_bot
```
